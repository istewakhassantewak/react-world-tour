# 🌍 React World Tour

A modern and responsive world tour web application built with **React.js** and **Vite**. This project allows users to explore different countries around the world with dynamic data fetching using the Fetch API.

## 🚀 Live Demo

🔗 **Live Website:** [React World Tour Live Demo](https://react-world-tour-rho.vercel.app?utm_source=chatgpt.com)

## 📂 GitHub Repository

🔗 Repository Link: [Repositroy Link](https://github.com/istewakhassantewak/react-world-tour/tree/main)

## 📌 Features

* 🌎 Explore countries around the world
* ⚛️ Built with React + Vite
* 📡 Dynamic data fetching using Fetch API

* ⚡ Fast performance with Vite

* 🔍 Interactive country information display

## 📦 Installation

Clone the repository and run the project locally.

```bash
# Clone the repository
git clone https://github.com/istewakhassantewak/react-world-tour.git

# Go to project folder
cd react-world-tour

# Install dependencies
npm install

# Start development server
npm run dev
```

---

## 🖥️ Build for Production

```bash
npm run build
```
